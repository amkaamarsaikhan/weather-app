Хотын нэр оруулж хайлт хийнэ үү.

Learning Points

Fetch API Basics

Fetch API Basics: Энэ нь вэб хөтөчөөс сервер рүү хүсэлт илгээх үндсэн хэрэгсэл юм. Энэ нь үргэлж Promise буюу "амлалт" буцаадаг.

Async/Await

Async/Await: Энэ нь олон дамжлагатай Promise-ийг илүү хялбар, яг л ердийн код шиг дарааллуулан уншигдахаар бичих боломжийг олгодог орчин үеийн арга юм.

Error Handling

Error Handling: API хүсэлт илгээх үед интернэт тасрах эсвэл буруу хот хайх зэрэг алдаа гарч болно. Тиймээс try/catch блок ашиглан алдааг ухаалгаар зохицуулах хэрэгтэйг сануулж байна.

PRACTICE EXERCISES

1. Хотын нэр хоосон үед алдаа шалгах (error handling) нэмэх
2.Ачаалж байгааг харуулах spinner (loading animation) нэмэх
3. Өөр API-оос өгөгдөл татах (жишээ нь: JSONPlaceholder)
4. Цаг агаарын нэмэлт мэдээлэл нэмэх (агаарын даралт, харагдах орчин гэх мэт)
5. Сүүлд хайсан хотыг localStorage-д хадгалах
6. Хайлт цэвэрлэх "Clear" товч нэмэх
7. Олон хотын цаг агаарын мэдээллийг нэг дор татаж харуулах
8. Буруу хотын нэр оруулахад алдаа боловсруулах
9. Огноо, цагийг форматлах функц үүсгэх
10. Цаг агаарын өөр өөр нөхцөлд тохирсон icon (дүрс) нэмэх
