Weather App

Enter a city name to search.

Learning Points

Fetch API Basics

A tool for sending network requests from the browser. It always returns a Promise.

Async/Await

Modern syntax that makes asynchronous code look and behave like synchronous code.

Error Handling

Using try/catch blocks to gracefully handle network issues or invalid search results.
